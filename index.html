Control Selectivo - Evaluación de Edad

Este ejercicio permite al usuario ingresar su edad. Dependiendo del valor ingresado, se muestra un mensaje que indica si el usuario es mayor o menor de edad. Si la edad es 18 años o más, se muestra "mayor de edad"; de lo contrario, "menor de edad".

Cálculo del Volumen de un Cilindro

Se desarrolla una página web que solicita al usuario ingresar la altura y el diámetro de un cilindro. A partir de estos datos, se calcula el volumen utilizando la fórmula adecuada y se presenta el resultado al usuario.

Valor de Venta de un Producto

Este ejercicio permite ingresar el valor de venta de un producto y calcula el Impuesto General a las Ventas (IGV) del 18%. También se muestra el precio total de venta, sumando el IGV al precio original.

Descuentos en un Restaurante

En este ejercicio, se permite al usuario ingresar el monto de su consumo en un restaurante. Se aplica un descuento del 10% para consumos de hasta S/.100.00 y un 20% para consumos mayores. Además, se calcula un impuesto del 18% sobre el total después del descuento, mostrando el monto del descuento, el impuesto y el total a pagar.

Descuentos según Tipo de Cliente: General y Afiliado

Este programa solicita al usuario ingresar el monto de su compra, junto con su tipo de cliente (general o afiliado) y la forma de pago (al contado o en plazos). Se calcula el descuento o recargo según una tabla predefinida, y se muestra el total a pagar, permitiendo al usuario entender mejor sus opciones de pago.
